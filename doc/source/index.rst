.. cpp:namespace:: onewiretemp

onewiretemp
===========

A C++ library for communicating with 1-wire temperature sensors. It should be compatible with:

- DS18S20
- DS1822
- DS18B20

which all use the same protocol.

You can find the code at https://github.com/tomjnixon/onewiretemp.
